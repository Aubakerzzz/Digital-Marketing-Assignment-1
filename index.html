Automated Data Pipeline — AWS + Airflow

Design and deploy a fully automated, cloud-native ETL pipeline ingesting multi-source business data into AWS S3, transforming it via Airflow DAGs, and loading it into a structured warehouse — running 24/7 without human intervention.
